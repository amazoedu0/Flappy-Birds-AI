# Flappy Bird AI [NEAT]
AI that plays Flappy Bird Game using the python module NEAT.

##  Instructions
Install Python Modules:
 `pip3 install -r requirements.txt`
 
 Simply run `main.py` and watch an AI start training itself to play the Flappy Bird Game!
